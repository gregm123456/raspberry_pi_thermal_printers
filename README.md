# Raspberry Pi Thermal Printers

Documentation and instruction sets for MC582H thermal printers.

Third-party dependencies:

- `python-escpos` as a git submodule at `third_party/python-escpos` (for ESC/POS
	printing support).

Note: The documentation files are manufacturer-supplied and may not be licensed
under the project's MIT `LICENSE`. See `DOCS_LICENSE.md` for details.
